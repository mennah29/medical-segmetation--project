High-Fidelity 3D Medical Segmentation & Interactive Evaluation

Analyzing Anatomical Segmentation Performance in CT Scans

Presenter: Mennah

Bridging Model Performance and Clinical Visualization

Goal

To move beyond simple segmentation metrics by building a comprehensive, interactive validation tool—a **Dash Application**—that allows researchers and clinicians to visually and quantitatively assess model performance on complex anatomical structures.

Methodology

1. Segmentation & Comparison

Utilizing predicted segmentation masks from a deep learning model and comparing them against Ground Truth data.

2. Metric Calculation

Generating key 3D overlap metrics: **Dice Score, IoU, and Volumetric Similarity.**

3. 3D Visualization

Converting volumetric data into interactive 3D meshes using the **Marching Cubes** algorithm.

4. Interactive Dash App

Integrating the visualization and metrics into a live, interactive environment.

Case Study 1: Analysis of Vertebrae Segmentation Fidelity

Key Findings (Quantitative)

  • ▶
    Dataset: Successfully processed 24 individual vertebrae segments.
  • ▶
    High Fidelity: Metrics show strong performance with Dice Scores from 0.78 to 0.94 and IoU from 0.64 to 0.88.
  • ▶
    Volumetric Consistency: High Volumetric Similarity (0.89 to 0.97) confirms strong agreement in volume.

Critical Insight

The application enables rapid identification of vertebrae with lower scores (e.g., S1 or C7), immediately pinpointing areas where the model requires further training.

Interactive Visual Validation: Dash App Features

  • Interactive Opacity Control: Adjust visibility for each label individually to "see through" predictions.
  • Real-Time Metrics: Evaluation scores are displayed alongside the 3D model for immediate context.
  • Robust 3D Engine: Handles complex mesh generation via Marching Cubes for multiple structures.

"This tool transforms static metrics into a dynamic, clinically relevant validation environment."

Next Steps & Project Impact

Key Conclusions

  • Interactive visualization is crucial for identifying segmentation failures that numeric metrics alone might mask.
  • The model demonstrated high overall performance, but specific segments (like certain vertebrae) require targeted investigation.

Future Directions

  1. Metric Expansion: Integrate Surface Distance Metrics (e.g., Hausdorff Distance).
  2. Multi-Structure Control: Expand the Heart app to control sub-structures (e.g., chambers).
  3. Performance Optimization: Optimize mesh generation for larger, high-resolution CT volumes.

Closing Statement

This project provides a reliable, interactive validation framework ready for deployment in a clinical research setting.
